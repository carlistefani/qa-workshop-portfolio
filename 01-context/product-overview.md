# Product Overview
## Nombre del producto
JPetStore

## Tipo de producto
Tienda online
Api de Mascotas

## Que problema resuelve
Los usuarios pueden acceder al catalogo actualizado de mascotas, asi tambien realizar compras de forma remota

## Como genera valor para el negocio
Los usarios pueden revisar el catalogo y realizar compras sin comprometer tiempo de atencion al cliente de forma presencial
La cobertura territorial del negocio se amplia
El comercio puede aumentar el caudal de clientes
La experiencia de compra onlina brinda comodidad y seguridad al cliente
